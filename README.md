# UserInfoManagement
一个极简化的JavaWeb项目 --- 用户信息管理系统

### 技术栈
Java + Mysql + Servlet + EL/JSTL + Jsp/css/Jquery + Tomcat8

### 工具
IntelliJ IDEA

### 主要功能
注册 + 登录 + 记住密码 + 自动登录 + 邮箱验证 + 分页 + 增/删/改/查用户信息
